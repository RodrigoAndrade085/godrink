# :beers: godrink

### :computer: Rodando Projeto:

  Clone o repositório:  
  `git clone https://github.com/RodrigoAndrade085/godrink.git`
  
  Acesse a pasta do projeto:  
  `cd godrink`
  
  Instale as dependências:  
  `yarn ou npm install`
  
  Rode o projeto com:  
  `yarn start ou npm run start`
