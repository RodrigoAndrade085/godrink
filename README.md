# :wrench: godrink

### :computer: Rodando Projeto:

  Clone o repositório:  
  `https://github.com/RodrigoAndrade085/refactoring-classes-ts.git`
  
  Acesse a pasta do projeto:  
  `cd refactoring-classes-ts`
  
  Instale as dependências:  
  `yarn ou npm install`

  Rode o server  
  `yarn server`
  
  Rode o front  
  `yarn start`
